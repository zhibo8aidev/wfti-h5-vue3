# WFTI 前端单体实现范围确认

- 时间：2026-04-27
- 来源：用户在 Feishu 群中的技术方案修订说明

## 结论
- 本项目 **不需要后端**，只做前端 H5。
- 16 道题题库、评分规则、隐藏人格检测、人格向量匹配、结果展示全部由前端处理。
- 本期不做结果上报、账号级存储、后端接口、服务端配置拉取。

## 对方案的影响
- 移除 backend implementation 范围与相关任务。
- `查看我的结果` 与断点续答改为依赖本地缓存 / 本地最近结果。
- 活动文案、二维码链接、人格数据、题库与算法规则都作为前端静态配置打包。
- 分享、保存相册、登录、站内跳转仍可通过宿主 bridge 适配，但不依赖后端。

## 当前基线
- 题库：前端静态配置
- 人格算法：前端本地计算
- 人格结果文案：前端静态配置
- 结果保存：仅本地缓存，不做后端持久化

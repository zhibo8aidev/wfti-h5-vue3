# WFTI — 世界杯球迷人格大鉴赏 设计方案

## 1. 核心视觉语言 (Visual Identity)

### 1.1 色彩规范
- **主色 (Primary):** `#0082FF` (直播吧品牌蓝，体现专业、赛事感)
- **辅助色:**
  - 成功/进球绿: `#00C853`
  - 警示/激情红: `#FF3D00`
  - 背景白: `#FFFFFF` (页面主基调)
  - 文字主色: `#1A1A1A`
  - 文字次色: `#666666`
- **渐变:** 建议在插画区和雷达图使用品牌蓝的淡化渐变，增加层次感。

### 1.2 字体规范
- **数字/代号:** 使用加粗的无衬线字体 (如 DIN Alternate 或 Roboto Bold)，强调力量感。
- **正文:** 优先使用系统默认无衬线字体 (PingFang SC, Microsoft YaHei)，确保 H5 环境渲染清晰。

### 1.3 组件规则
- **圆角:** 核心卡片和按钮采用 `8px` 或 `12px` 圆角，平衡力量感与亲和力。
- **阴影:** 采用轻量化的投影 (`0 4px 12px rgba(0,0,0,0.05)`)，保持页面简洁。
- **按钮 (CTA):** 
  - 落地页/结果页底部固定悬浮按钮，宽度占据屏幕约 90%，高度 `48px`-`52px`。
  - 状态：Normal, Pressed (缩放 0.98), Disabled (置灰)。

## 2. 页面结构与布局 (Page Architecture)

### 2.1 落地页 (Home)
- **首屏构图:** 顶部品牌 Slogan + 中央视觉主体 (世界杯氛围插画) + 底部 CTA。
- **沉浸感:** 背景可带淡淡的足球场底纹。

### 2.2 答题页 (Quiz)
- **进度条:** 位于顶部，`4px` 高度，`#0082FF` 填充。
- **题干区:** 居中对齐，大字展示，确保阅读效率。
- **选项区:** 纵向排列，点击反馈明显 (背景变蓝色，文字变白)。

### 2.3 计算过渡页 (Transition)
- **视觉中心:** 动态扫描动效，球场战术线或 3D 旋转足球。
- **文案:** 底部文字淡入淡出。

### 2.4 结果页 (Result) - 核心
- **人格卡片:** 
  - 顶部代号 (如 GOAT) 使用大号空心或渐变字。
  - 插画区：根据 16 种人格对应不同的人物剪影或插画。
  - 标签云：围绕插画散落排列，字体大小错落。
- **维度雷达图:** 8 轴分布，`#0082FF` 半透明填充，展示用户 DNA。
- **关系链:** 球星对照、天敌、拍档采用小头像 + 昵称形式，横向排布。

## 3. 交互动效 (Interaction)

- **切题动效:** 当前题容器 `translateX(-100%)` 消失，新题 `translateX(0)` 从右侧 `100%` 划入，时长 `400ms`, `ease-in-out`。
- **点击反馈:** 按钮点击瞬间缩小，选项点击高亮 `300ms`。
- **加载状态:** 骨架屏优先，加载超过 3s 使用球类旋转 Loading。

## 4. 长屏 Mobile-Web 适配注意事项

- **安全区域:** 底部固定按钮需避开 iOS Home Indicator。
- **背景连续性:** 结果页内容较长，背景需采用渐变或无缝拼贴纹理。
- **首屏锁定:** 落地页和答题页锁定单屏高度，禁止滚动；结果页允许滚动但需固定底部操作栏。

## 5. 分享海报设计 (Poster)

- **尺寸:** 750px * 1334px (适配主流手机)。
- **布局:** 
  - 顶部 2/3 展示人格核心信息（代号、插画、雷达图）。
  - 底部 1/3 展示解析文案、二维码、Logo。
- **二维码:** 居右下角，左侧搭配“长按识别测试你的球迷人格”引导语。

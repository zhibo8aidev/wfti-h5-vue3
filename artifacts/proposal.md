# 世界杯球迷人格大鉴赏（WFTI）H5 活动 Proposal

## 项目背景
2026 世界杯赛期（6 至 7 月）是直播吧赛事流量峰值窗口。本项目拟上线一个嵌入直播吧 App 的手机端 H5 互动活动，以“16 题测球迷人格”为核心玩法，承接流量、强化表达欲与分享意愿，并反哺站内专题、社区与竞猜场景的活跃度。

## 项目目标
- 活动期形成高参与测试玩法，提升赛事大盘互动率
- 通过人格结果页与海报分享，提升用户自传播与社交讨论度
- 通过结果持久化与站内联动，带动专题、社区、竞猜等模块回流

## 目标用户
- 世界杯期间高频浏览直播吧赛事内容的球迷用户
- 对人格测试、标签化表达、结果分享有兴趣的轻互动用户
- 喜欢在社区表达观点、参与话题讨论和竞猜互动的用户

## 方案概述
### 1. 产品形态
- 手机端 H5，运行于直播吧 App WebView
- 前端技术栈明确为 Vue 3
- 页面视觉基调为白色背景，品牌主色为 `#0082FF`，整体营造世界杯氛围

### 2. 核心玩法
- 用户进入活动落地页后，完成固定顺序的 16 道题测试
- 前端本地完成 8 维评分、隐藏人格优先检测、16 种常规人格向量匹配
- 生成人格结果页，支持查看解析、雷达图、球星对照、最佳拍档 / 天敌及二维码信息
- 用户可生成分享海报，并通过宿主分享面板分发或保存到相册

### 3. 页面范围
- 活动落地页
- 答题页
- 结果计算过渡页
- 结果页
- 分享海报生成态
- 活动异常状态页（未开始 / 已结束 / 网络异常）

### 4. 数据与联动范围
- 结果生成后异步上报 UID、人格代号、原始分与时间戳
- 同一用户重新测试覆盖旧结果
- 为后续站内专题入口、社区标签、竞猜彩蛋、个人主页人格标签预留消费边界

## MVP 范围
### 包含
- 16 题主链路答题体验
- 24 小时断点续答
- 前端人格计算引擎
- 结果页完整展示与异步持久化
- 海报生成、系统分享、保存相册
- 移动端异常与活动状态兜底页
- 宿主桥接与结果服务的数据边界定义

### 不包含
- 新增客户端原生能力开发
- 人格详情二级页或人格百科体系
- 复杂后台运营配置系统
- H5 站外独立传播页改造

## 关键产品决策
- 前端实现采用 Vue 3，避免偏离 H5 目标形态
- 设计目标为 mobile-web，后续需输出 HTML 高保真 mockups
- 题库、人格结果文案、算法规则作为前端静态配置基线固化
- 结果计算全部在前端完成，后端仅承担持久化与联动数据输出
- 宿主桥接采用 adapter 边界，减少对未确认协议的耦合

## 关键风险
- 宿主登录、分享、保存相册、路由跳转能力需尽快确认桥接协议
- 海报生成依赖 Canvas / html2canvas，低端设备存在性能与兼容性风险
- 人格结果页的视觉感染力强依赖设计阶段插画、标签云和氛围图形质量
- 站内联动涉及多方协作，需尽快明确消费字段与上线窗口
